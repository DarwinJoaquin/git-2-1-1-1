Perfil de egreso

Información sobre el perfil de egresado de la especialidad


La información que ofrece la carrera de Técnico en Programación permite al egresado, a través de la articulación de saberes de diversos campos, realizar actividades dirigidas a la :

  • Instalación y desarrollo de software de aplicación utilizando programación estrucutrada y orientada a objetos en ambientes web y móviles, con alamacenamiento persistente de datos,
  • Así como la configuración y administración de plataformas e-learning y
  • comercio electronico
Durante el proceso de formación de los cinco módulos, el estiudiante desarrollará o reforzará
Las siguiente competencias profesionales:
  • Desarolla e instala software de aplicación utilizando programación estructurada,con almacenamiento persistentre de datos
  • Desarrolla software de aplicación utilizando programación orientada a objetos con almacenamiento persistente de los datoa
  • Desarrolla aplicaciones web y móviles
  • Administra sistemas operativos,aplicaciones y servicios
  • Desarrolla, administra y configura soluciones de e-learning y comercio electrónico

  • Modulo I

    Desarrolla e instala software de aplicación utilizando programación estructurada, con almacenamiento persistente de datos

  • Modulo II

    Desarrolla software de aplicación utilizando programación orientada a objetos, con almacenamiento persistente de datos

  • Modulo III

    Desarrolla aplicaciones web y móviles

  • Modulo IV

    Desarrolla sistemas operativos, de aplicaciones de servicio

  • Modulo V

    Desarrolla, administra y configura soluciones de e-learning y comercio electrónico
